# CHECK-IN  
AI-Powered Mental Wellness Companion  

---

## Overview  
CHECK-IN is an AI-driven mental wellness application designed to support individuals dealing with stress, loneliness, and emotional challenges.  

The platform focuses on healing, self-awareness, and emotional support rather than medical treatment. It provides users with a safe and private environment to understand their mental state through continuous analysis and intelligent interaction.

---

## Problem Statement  
A large number of individuals today face:
- Stress, anxiety, and depression  
- Emotional isolation and loneliness  
- Fear or hesitation in consulting psychologists  
- Lack of continuous mental health monitoring  

There is a need for a system that provides a non-judgmental and accessible support mechanism.

---

## Solution  
CHECK-IN offers:
- AI-based emotional analysis  
- Continuous behavioral monitoring (with user consent)  
- Gamified engagement for habit improvement  
- A personalized AI companion for interaction  

The system acts as a supportive companion and does not replace professional medical advice.

---

## Features  

### Mental State Analysis  
- Animated onboarding questions based on psychological factors  
- PHQ-9 based assessment  
- Classification using K-Nearest Neighbors (KNN)  
- Personalized insights stored securely  

---

### Forest Growth System  
- Uses YOLO (You Only Look Once) for real-time detection  
- Tracks user attention, posture, and activity  
- Daily interaction contributes to virtual forest growth  
- Visual representation of user consistency and engagement  

---

### AI Companion  
- Customizable virtual assistant  
- Natural Language Processing (NLP) based interaction  
- Emotion-aware conversational responses  
- Learns user behavior over time  

---

### Smart Journal  
- Free-form journaling system  
- Sentiment and language analysis using NLP  
- Deeper insights generated using Large Language Models (LLMs)  

---

### Relaxation Games  
- Interactive mini-games designed to reduce stress  
- Improves cognitive focus and engagement  

---

### Healing Module  
- Personalized recommendations including:
  - Meditation  
  - Yoga  
  - Lifestyle improvements  
- Suggestions based on user behavior and analysis  

---

### 30-Day Monitoring System  
- Continuous tracking with user permission:
  - Typing patterns  
  - Screen usage  
  - App switching behavior  
  - Late-night activity  
  - Music listening repetition  

---

### Trusted Contact Feature  
- Users can assign a trusted contact  
- Summary reports can be shared with consent  

---
